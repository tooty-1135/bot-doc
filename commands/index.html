    指令大全

    這裡列出了所有的指令，詳細看 這裡

    dvc

    動態語音頻道

    /dvc setup

    初始化動態語音頻道
    使用方式: dvc setup

    /dvc setlimit

    設定動態語音頻道的最大人數
    使用方式: dvc setlimit [人數]

    /dvc lock

    鎖定你的語音頻道
    使用方式: dvc lock

    /dvc unlock

    解鎖你的語音頻道
    使用方式: dvc unlock

    /dvc permit

    允許某個人進入你的語音頻道
    使用方式: dvc permit [@一個人]

    /dvc reject

    拒絕某個人進入你的語音頻道
    使用方式: dvc reject [@一個人]

    /dvc limit

    設定自己動態語音頻道的最大人數
    使用方式: dvc limit [人數]

    /dvc name

    更改你的語音頻道名稱
    使用方式: dvc name [名稱]

    fun

    好玩的東西

    /8ball

    一個8ball?
    使用方式: 8ball [你要問的問題]

    /choice

    choice
    使用方式: choice [要選的東西，用空格分開]

    level

    等級系統

    /rank

    查看等級
    使用方式: rank

    main

    雜項

    /ping

    查看機器人的延遲
    使用方式: ping

    /say

    照著說某段話
    使用方式: say [要說的話]

    /avatar

    拿到某個人的頭像
    使用方式: avatar [@某個人]

    /time

    查看現在時間
    使用方式: time

    /server

    關於伺服器
    使用方式: server

    /user

    關於某個人
    使用方式: user [@某個人]

    /bot

    關於這個機器人
    使用方式: bot

    /test

    test
    使用方式: test

    money

    金錢系統

    /new

    建立帳號
    使用方式: new

    /sign

    每日遷到
    使用方式: sign

    /money

    查看有多少錢
    使用方式: money [@某個人]

    /redeem

    兌換代碼
    使用方式: redeem [代碼]

    /payment

    要求付款
    使用方式: payment [@某個人] [要付多少錢]

    music

    音樂系統

    /pause

    暫停/撥放音樂
    使用方式: pause

    /volume

    改變音量(0~250)
    使用方式: volume [音量(0~250)]

    /skip

    跳過歌曲
    使用方式: skip

    /queue

    顯示當前播放清單
    使用方式: queue

    /stop

    停止播放
    使用方式: stop

    /play

    播放
    使用方式: play [網址或搜尋]

    ticket

    客服單系統

    /rank

    建立一個客服單的按鈕
    使用方式: ticket [#頻道]

    verify

    驗證系統

    /verify setup

    初始化驗證系統
    使用方式: verify setup [#頻道] [@身分組]

    welcomer

    歡迎訊息

    /welcomer

    初始化歡迎訊息
    使用方式: welcomer [#頻道]
